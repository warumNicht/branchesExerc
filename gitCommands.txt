
git add <file> to include in what will be committed

git reset HEAD <file>  to unstage

git checkout -- <file> to discard changes in working directory

git diff показва неиндексираните разлики (всичко по-ново от последния комит)

git log

git commit -a -m "message" индексира всичко ново и комитва наведнъж


git config --global user.name "your username"

git config --global user.password "your password"
 

git branch <branch name> създаване на нов клон. Това създава нов указател към същия къмит, на който сте в момента

git log --decorate   отпечатва накъде сочат указателите на разклоненията.

git checkout <branch>  превключва към клон


git log --oneline --decorate --graph --all графика на клоновете
